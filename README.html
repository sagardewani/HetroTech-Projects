  • Transition Plugin is a jQuery and Javascript based plugin to dynamically set hover transitions on HTML elements.
  • This plugin provide you easy and convenient way to apply hover effect on element.
  • This plugin comes with many handy features that are just one press away from you. Just press the command keys and see the magic of plugin.
  • This plugin is intended to apply hover transitions effect and generate CSS stylesheet classes so that you can just copy and paste the code of CSS in your HTML file to see the effect.
  • This plugin is for testing before implementing your CSS styles into HTML. For actual effect you must copy the CSS code generated and paste it in your HTML file.
  • Now,This plugin is not compaitible with mobile browser versions.

HOW TO SETUP Transition Plugin

//////////////////////////////////////////
  • To setup Transition plugin for your project you should include the following plugins as dependency in your project before including this plugin:
    The plugin is tested with mentioned dependencies
    • jQuery 3.x
    • Bootstrap 3.7.x
    • jQuery UI 1.11.x
    • font-awesome 4.7.x
  • After including all these dependencies insert Transition plugin javascript file in your project using :
    <script type="text/javascript" src="jquery.transition.min.js" ></script>
  • How To Use
    $.hoverTransition();
    <script> $.hoverTransition(); </script>
    You can also change the triggering key combination
    Option Example Usage Description
    modeKey 
    $.hoverTransition(modeKey:'66');
    		 
    $.hoverTransition(modeKey:'your custom key decimal value');
    		 This will customized the default key combination for triggering mode alert that is ctrl+m to ctrl+b
    Note:this doesn't change the control key(i.e. shift/alt/ctrl).Only the combination with control keys.
    sourceKey 
    $.hoverTransition(sourceKey:'66');
    		 
    $.hoverTransition(sourceKey:'your custom key decimal value');
    		 This will customized the default key combination for triggering CSS styles code that is Enter to b
    transitionKey 
    $.hoverTransition(transitionKey:'66');
    		 
    $.hoverTransition(transitionKey:'your custom key decimal value');
    		 This will customized the default key combination for triggering transition that is shift+h to shift+b
    Note:this doesn't change the control key(i.e. shift/alt/ctrl).Only the combination with control keys.
  • How to make an element disable to be selected

    You just need to add a class(d-set) to that element to make unavailable to be selected

    Example: <input class = "d-set" />

Transition Plugin FUNCTIONALITIES

    Transition Plugin comes with varities of functionality that are listed below that can be useful to boost the designing of your project.
    #### MODES ####
    Transition Plugin have 1 mode
    # Mode Name #Key Combination #Description #Procedure
    1 Set Transition Shift+h To Set onhover transition on the selected element There could be consequences of applying transition so you should have some knowledge how transitions are applied If you have already applied some effect then there might be no effect of these transitions on the elements
    1. First Select The Element on which you want to apply onhover effect/transition
    2. Second Press the key combination.
    3. Third A box will open to set the transition configuration.
      # Transition Name Effect Procedure
      1 none Remove all applied hover transition(no effect on transitions already applied.) Select the none from Select Transition type menu.
      2 All To apply your custom transition(you should have knowledge of transitions for this.)
      1. Select the all from Select Transition type menu.Then press
      2. Enter the transition duration.If you don't know about this just press
          To set Custom transition duration
        • Click on edit icon below the input box. To enable the input button
        • Enter the custom duration and then press
      3. Enter the transition name(if you don't know refer to CSS transitions first). Then press
        • if you want to change more than one property after hover then separate them with ,
        • Example: background-color,,color,,width,,etc...
      4. Enter the transition value(if you don't know refer to CSS transitions first). Then press
        • if you want to change more than one property value after hover then separate them with ,
        • Example: white,,red,,500px,,etc...
        • Note: you may use !important to see the changes cause of overriding of styles can prevent the applied effect. However, I would not suggest using !important in your actual HTML layout. I have used it so that it can override the inline or higher priority CSS style properties to show the effects.
      5. Select the transition mode and press
        • Auto : this set "ease-in-out" transition
        • Linear : this set "linear" transition
        • Ease: this set "ease" transition
      3 Fade To apply fade transition
      1. Select the fade from Select Transition type menu.Then press
      2. Enter the transition duration.If you don't know about this just press
          To set Custom transition duration
        • Click on edit icon below the input box. To enable the input button
        • Enter the custom duration and then press
      3. Enter the opacity value. If you don't know about this just press
          To set Custom opacity value
        • Click on edit icon below the input box. To enable the input button
        • Enter the custom opacity value and then press
      4. Select the transition mode and press
        • Auto : this set "ease-in-out" transition
        • Linear : this set "linear" transition
        • Ease: this set "ease" transition
      4 Size To apply size transition.This transition will change the size of element.
      1. Select the size from Select Transition type menu.Then press
      2. Enter the transition duration.If you don't know about this just press
          To set Custom transition duration
        • Click on edit icon below the input box. To enable the input button
        • Enter the custom duration and then press
      3. Enter the width value in numeric(scale is px). If you don't know about this just press
          To set Custom width value
        • Click on edit icon below the input box. To enable the input button

      4. Select the transition mode and press
        • Auto : this set "ease-in-out" transition
        • Linear : this set "linear" transition
        • Ease: this set "ease" transition
      5 Move To apply move transition. This transition will change the position of element.
      1. Select the move from Select Transition type menu.Then press
      2. Enter the transition duration.If you don't know about this just press
          To set Custom transition duration
        • Click on edit icon below the input box. To enable the input button
        • Enter the custom duration and then press
      3. Enter the distance value(or movement value) in numeric(scale is px). This value must be set. Then press
      4. Select the movement direction and press
        • Left : move the element by distance value in left direction.
        • Right : move the element by distance value in right direction.
      5. Select the transition mode and press
        • Auto : this set "ease-in-out" transition
        • Linear : this set "linear" transition
        • Ease: this set "ease" transition
